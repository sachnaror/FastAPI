# fastapi-example

A simple example of using Fast API in Python.

## Preconditions:

- Python 3

## Clone the project

```
git clone https://github.com/marciovrl/fastapi-example.git
```

## Install dependencies

```
pip install -r requirements.txt
```

## Run server

```
uvicorn main:app --reload
```

## Run test

```
pytest test.py
```
